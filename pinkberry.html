Camera Hunt Website

The Problem:
With thousands of different options, purchasing a camera can be overwhelming for individuals with differing levels of expertise.
The Solution:
CameraHunt is a hypothetical website created to help users find a camera that best fits their needs. To make the camera hunting process easier, I created a step-by-step approach that prompts users with questions which help to decide what features each user finds most important to have in their ideal camera.
Designs


The website starts on the landing page of CameraHunt. The website uses information about the users interests to select a camera. It makes choosing a camera type much simpler by narrowing down the styles based off of previous answers. Once a user answers all questions, the website ends with a page that shows the user three cameras options it found to be most fitting given their response to the questions.
